# Red Wine Quality Prediction

## Overview
This project predicts the quality of red wine based on its chemical properties.

## Project Structure
- `all_ml_moedl.ipynb`: Jupyter notebook for model training and comparison.
- `winequality-red.csv`: Dataset with red wine attributes and quality scores.
- `saved_models/`: Directory containing exported models.

## Usage
Run the `all_ml_moedl.ipynb` notebook to see the results of various machine learning models.
