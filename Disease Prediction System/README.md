# Disease Prediction System

## Overview
This repository contains a disease prediction system focusing on Breast Cancer prediction using machine learning.

## Project Structure
- `Brest_Cancer_Prediction.ipynb`: Jupyter notebook for model training and evaluation.
- `Brest_cancer_data.csv`: Dataset used for training.
- `cancer.pkl`: Saved model file.
- `archive/`: Compressed data or older versions.

## Usage
Run the `Brest_Cancer_Prediction.ipynb` notebook to see the analysis and prediction results. Ensure you have libraries like `scikit-learn` and `pandas` installed.
