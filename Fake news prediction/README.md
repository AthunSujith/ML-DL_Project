# Fake News Prediction

## Overview
A machine learning project that analyzes news articles to determine their authenticity, classifying them as either 'True' or 'Fake'.

## Project Structure
- `ml_moels.ipynb`: Jupyter notebook for data preprocessing and model training.
- `True.csv` & `Fake.csv`: Datasets containing authentic and fabricated news.
- `Final_data.csv`: Merged and preprocessed data.
- `saved_models/`: Directory where trained models are stored.

## Usage
Run `ml_moels.ipynb` to train and evaluate the models.
