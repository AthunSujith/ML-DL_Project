# Spam Mail Prediction

## Overview
This project uses Natural Language Processing and Machine Learning to classify emails as spam or ham (not spam).

## Project Structure
- `all_ml_models.ipynb`: Notebook for data cleaning, preprocessing, and model training.
- `mail_data.csv`: Dataset containing raw email text and labels.
- `spam_model.pkl`: The trained classification model.
- `tfidf_vectorizer.pkl`: The saved TF-IDF vectorizer used for text processing.

## Usage
Run the `all_ml_models.ipynb` notebook to see the classification results.
